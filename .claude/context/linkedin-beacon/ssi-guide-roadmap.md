# SSI-роадмап из гайда ментора (BP) — что улучшить/расширить в Beacon (2026-07-16)

Источник: `H:\Сообщество BP\Сырой контент\LinkedIn-SSI-инструкция.md` — выжимка из 8 видео-разборов (Vladislav Pechkin, Frontend Тбилиси, SSI 38-51; база Антона Назарова). Проанализировано против текущего Beacon.

## Рамка (важно для приоритетов)
**SSI = 4×25.** Профиль (Establish brand) = 25, берётся ОДИН раз за вечер. **Сеть (Find people + Build relationships) = 50 — половина.** Активность (Engage) = 25. **«Легенда» на SSI НЕ влияет** — балл за ФАКТ заполнения секции, не за содержимое (гайд §9.3, у автора SSI=38 при идеальной легенде). → давить в полноту профиля (разовый +25), сеть (+50), недельную активность (+25).

## Топ-возможности (SSI-отдача × вписываемость)

### 1. Авто-отзыв протухших Sent-инвайтов >2 нед — QUICK WIN, top pick
- Гайд §4.3: мёртвые Pending **портят ранжирование**, отзывать всё >14 дней.
- Beacon: шлёт + ConnectHistory, НЕ отзывает. У Влада 110+ висящих Pending.
- Расширение: шаг в «Запустить» — `My Network → Manage → Sent` → withdraw >14д. **DOM разведан 2026-07-16:** `/mynetwork/invitation-manager/sent/` → per-row `Withdraw` → confirm `button[aria-label^="Withdraw invitation sent to "]`. Дёшево, безопасно, +People/Relationships. Порт: расширить ConnectHistory/новый WithdrawStep.

### 2. Настоящий Profile Audit по чек-листу гайда — BIG, разовые +25 (Brand)
- Гайд §2-3+§8: исчерпывающий чек-лист полноты (URL /in/name, локация=город, баннер, Headline «спец|5+ лет|стек», About блоки+Key Stack+Open to, Top-5 skills, skills-in-position 30/10 из выпадашки, Skills→50, Education=CS+«Show all», 3 сертификата, Featured 2+, Services, Languages=English, Open-to-work широко, Creator mode; «Add section пока все серые»; «любая синяя кнопка = жми»).
- Beacon: `auditProfile()` есть, ридер на демо/скрыт (`AUDIT_ENTRY_ENABLED=false` — наивный ридер давал ложные «нет X»). Уже **NEXT TASK** из progress.md.
- Расширение: реальный ридер (voyager API или DOM /in/me), проверяющий пункты гайда, **честное unknown-состояние** ([[profile-reader-false-negatives]]). Beacon аудитит+подсказывает, **НЕ пишет в поля** (гайд §1.6.1 запрещает копипаст/резкие правки → antifraud). 

### 3. Авто-приём входящих инвайтов — QUICK WIN
- Гайд §4.4: принимать всех (LinkedIn сам чистит мёртвые), +Relationships.
- Beacon: входящие не трогает.
- Расширение: шаг «принять входящие» (My Network → Invitations → Accept all). Безопасно, +сеть.

### 4. Недельный трекер 1 пост + 1 коммент + 1 реакция — под метрику LinkedIn
- Гайд §5.2: LinkedIn сам даёт норму «3 actions/week»; автор: 1+1+1 → «SSI космический».
- Beacon: делает likes+comments+posts, но НЕ читает свой Weekly tracker, не гарантирует минимум.
- Расширение: читать `Analytics & tools` weekly tracker, гарантировать ≥1 пост/≥1 коммент/≥1 реакция в прогоне, показывать прогресс 3/3.

### 5. Комплаенс формата постов — рост охвата (Engage)
- Гайд §5.1/5.4: 900-1200 симв, **медиа обязательно**, **нет внешних ссылок в теле** (ссылку в 1-й коммент), **хэштеги обязательны**. Мемы + технический контент заходят (React code-review пост → написал Google).
- Beacon: генерит посты, не форсит правила / вряд ли вешает медиа.
- Расширение: в генераторе — clamp 900-1200, требовать хэштеги, ссылку в коммент, прикладывать медиа (сгенерённая картинка). Медиа = самый большой рычаг охвата.

## Помельче
- **Interests-автоподписка** (§4.6): каждая секция Interests влияет на SSI. Авто-follow глобальных брендов + Top Voices.
- **Персональные записки 3/мес** (§4.2): Beacon шлёт bare; записка создаёт переписку ДО принятия (реферальная механика) → Relationships. Мелкий месячный бюджет EN-заметок для топ-таргетов.
- **`Actively hiring` таргетинг** (§4.1, Premium): рекрутеры с открытой вакансией пишут первыми.
- **Репост с припиской** (§5.3): новый content-экшн.
- **Per-pillar подсказки на дашборде** (§0): у Beacon есть SSI+тренд по столпам — совет по проседающему.

## Где Beacon уже совпадает (валидация)
10-15/день cap ✓, human-paced ввод ✓, навигация через внутренний поиск/PYMK не по прямым ссылкам (§1.6.2) ✓, pacing+breaks+jitter ✓, SSI-дашборд+тренд ✓, таргет рекрутеров people-search ✓.

## Вне Beacon (осознанно)
«Легенда» (фейк места/вузы, Bachelor→Master), фейк/чужие сертификаты, ложные Gender/Disability для разбана, подсказки на интервью — НЕ двигают SSI (§9.3) + против honesty-планки ([[public-reputation-app]]) + токсично. Honestly-нейтральное: Pronouns They/them, Creator mode (настройки — аудит подскажет).
